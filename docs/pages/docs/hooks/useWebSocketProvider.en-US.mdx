# useWebSocketProvider

Hook for accessing the `Client`'s ethers.js WebSocket Provider.

```ts
import { useWebSocketProvider } from 'wagmi'
```

## Usage

```tsx
import { useWebSocketProvider } from 'wagmi'

function App() {
  const webSocketProvider = useWebSocketProvider()
}
```

## Return Values

```tsx
ethers.WebSocketProvider
```
